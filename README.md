# 📉 Customer Churn Prediction

## 📌 Overview
This project analyzes customer data to predict churn using machine learning models and uncover key factors influencing customer retention.

## 🎯 Objectives
- Perform data preprocessing and cleaning  
- Train classification models  
- Evaluate model performance  
- Identify key drivers of churn  

## 🛠️ Tools
- Python (pandas, scikit-learn)  
- seaborn, matplotlib  

## 🤖 Models Used
- Logistic Regression  
- Random Forest  

## 📊 Results
- Random Forest achieved higher accuracy  
- Key churn drivers include customer service calls and usage patterns  

## 💡 Insights
- Customer dissatisfaction strongly influences churn  
- Behavioral data can predict churn effectively  

## 📌 Conclusion
Machine learning can help businesses proactively reduce churn and improve retention strategies.
